Mathematics › Linear Algebra › Vector Spaces & Linear Transformations
Linear algebra is the study of vector spaces and linear transformations between them. It provides the mathematical language for geometry, physics, optimization, and nearly all of modern machine learning — from linear regression to deep neural networks. This reference builds from first principles: we begin with the axioms of a vector space and proceed through subspaces, bases, linear transformations, determinants, and eigenvalues, with each claim stated as a formal definition, theorem, or lemma accompanied by its proof.
A1.1

Power Set

For any set A, the power set of A, denoted P(A), is the set whose elements are all subsets of A. Formally,

P(A) = { B ∣ B ⊆ A }.

If A has n elements, then P(A) has 2n elements.

  • P(A)the power set of A — the collection of all subsets of A.
A1.2

Set of Functions

If A and B are two sets, the notation BA refers to the set of all functions from A to B:

BA = { f : A → B }.

Each function f ∈ BA assigns to every element of A exactly one element of B.

  • BAthe set of all functions f : A → B.
A1.3

Real‑Valued Function Space

When the codomain is the field of real numbers ℝ, the set ℝA is the space of all real‑valued functions on A. This set becomes a vector space with pointwise addition and scalar multiplication:

(f + g)(x) = f(x) + g(x),  (α f)(x) = α · f(x)

for all f, g ∈ ℝA, α ∈ ℝ, and x ∈ A.

  • ℝAthe vector space of real‑valued functions on A.
A1.4

Cardinality of a Finite Set

For a finite set A, the cardinality |A| is the number of elements in A. If A is finite, then ℝA can be identified with the familiar coordinate space ℝ|A| by enumerating the elements of A.

  • |A|cardinality (number of elements) of the finite set A.
  • ℝ|A|the standard coordinate space of dimension |A|, isomorphic to ℝA when A is finite.
A1.5

Indicator Function

For a subset C ⊆ A, the indicator function of C is the function 1C : A → {0,1} defined by

1C(x) = { 1 if x ∈ C,
     0 if x ∉ C. }

The notation 1x∈C is sometimes used as a compact equivalent of 1C(x).

  • 1Cindicator function of the set C; maps any x to 1 if x ∈ C, else 0.
  • 1x∈Calternative compact notation for 1C(x).
A1.6

Euclidean Space and Coordinate Notation

The d-dimensional Euclidean space ℝd consists of all ordered d-tuples of real numbers. Its elements (called vectors) are usually written as column vectors with coordinates indexed by parenthesized exponents:

x = (x(1), x(2), …, x(d))T.

The parentheses around the index distinguish a coordinate from a power. Thus x(i) denotes the i-th coordinate, not the i-th power of x.

  • ℝdthe d-dimensional Euclidean space.
  • x(i)the i-th coordinate of vector x, i = 1,…, d.
A1.7

Points, Vectors, and the Affine Viewpoint

Geometrically, ℝd can be seen in two ways:

  • As an affine space: its elements are points — fixed locations.
  • As a vector space: its elements are vectors — directions and magnitudes that can be added and scaled.

After choosing the origin 0d, every point can be identified with the vector from the origin to that point. In this text we will not distinguish notationally between points and vectors; both are treated as elements of the vector space ℝd.

  • affine spacea set of points; a point is a position.
  • vector spacea set of vectors; a vector is a translation (direction + magnitude).
A1.8

Zero Vector, Ones Vector, and Identity Matrix

Three fundamental objects are given special notation:

  • Zero vector 0d: the vector in ℝd with all coordinates 0.
  • Ones vector 1d: the vector in ℝd with all coordinates 1.
  • Identity matrix Id: the d × d matrix with 1 on the main diagonal and 0 elsewhere.
  • 0dthe d-dimensional zero vector.
  • 1dthe d-dimensional ones vector.
  • Idthe d × d identity matrix.
A1.9

Canonical Basis

The columns of the identity matrix Id form the canonical basis (or standard basis) of ℝd. They are denoted e1, e2, …, ed, where ei has a 1 in the i-th coordinate and zeros elsewhere:

ei = (0, …, 0, 1, 0, …, 0)T (1 at position i).

Every x ∈ ℝd can be uniquely expressed as x = ∑i=1d x(i) ei.

  • eithe i-th standard basis vector, i = 1,…, d.
A1.10

Euclidean Norm (ℓ2 Norm)

The Euclidean norm (or ℓ2 norm) of a vector x ∈ ℝd is denoted |x| and defined by

|x| = √( (x(1))2 + (x(2))2 + ⋯ + (x(d))2 ).

It satisfies the axioms of a norm: positive definiteness, homogeneity, and the triangle inequality. The notation ∥x∥2 is also used.

  • |x|the Euclidean (ℓ2) norm of x.
  • ∥x∥2alternative notation for the Euclidean norm.
A1.11

ℓp Norm for p ≥ 1

For any real number p ≥ 1, the ℓp norm of x ∈ ℝd is defined by

∥x∥p = ( |x(1)|p + |x(2)|p + ⋯ + |x(d)|p )1/p.  (1.1)

For every p ≥ 1, ∥·∥p satisfies the triangle inequality (Minkowski’s inequality) and therefore is a genuine norm. Important special cases:

  • p = 1: Manhattan (or ℓ1) norm.
  • p = 2: Euclidean norm (|x|).
  • p → ∞: maximum norm (the largest absolute coordinate).
  • ∥x∥pthe ℓp norm of x, p ≥ 1.
A1.12

ℓp for 0 < p < 1 and the ℓ0 “Norm”

For 0 < p < 1 the formula (1.1) remains well‑defined, but it does not define a norm: the triangle inequality fails. Despite this, the family is used in the study of sparsity because the limit

∥x∥0 ≝ limp→0 ∥x∥pp = limp→0 ∑i=1d |x(i)|p

counts exactly the number of non‑zero coordinates of x. This quantity is called the ℓ0 “norm” (though it is not a norm). It is the natural measure of sparsity.

  • ∥x∥p (0 < p < 1)a non‑norm quantity; fails the triangle inequality.
  • ∥x∥0the number of non‑zero entries of x; the ℓ0 “norm”.
A1.13

Norm Notation Conventions

Throughout the text we follow these conventions:

  • Single bars |·| are used for norms of finite‑dimensional vectors (typically the Euclidean norm).
  • Double bars ∥·∥ are reserved for norms of infinite‑dimensional objects, such as functions in Hilbert spaces.
  • The notation |A| with a set A denotes the cardinality of A; the meaning (vector norm vs. set size) is always clear from context.
  • |·|norm on a finite‑dimensional space.
  • ∥·∥norm on an infinite‑dimensional space.
  • |A|cardinality of the set A.
A1.14

Matrix Spaces

The set of all m × d real matrices (i.e., matrices with m rows and d columns, all entries real) is denoted Mm,d(ℝ), or simply Mm,d. When m = d we also write Md for the space of square matrices of size d × d.

The subset of d × d matrices that are invertible (i.e., have a two‑sided multiplicative inverse) is the general linear group, denoted GLd(ℝ):

GLd(ℝ) = { A ∈ Md ∣ there exists A−1 ∈ Md with A A−1 = A−1 A = Id }.
  • Mm,d(ℝ)the set of all real m × d matrices; often just Mm,d.
  • Mdshorthand for Md,d, the set of d × d square matrices.
  • GLd(ℝ)the general linear group: the set of invertible d × d real matrices.
A1.15

Matrix Construction from Column Vectors

Given m column vectors x1, …, xm ∈ ℝd, the notation [x1, …, xm] stands for the d × m matrix whose j-th column is xj. Thus the d × d identity matrix can be written as

Id = [e1, e2, …, ed],

where e1, …, ed are the canonical basis vectors.

  • [x1, …, xm]the d × m matrix formed by horizontally stacking the column vectors x1, …, xm.
A1.16

Matrix Entries, Rows, and Columns Notation

For a matrix A ∈ Mm,d(ℝ), the entry in row i and column j is denoted either A(i, j) or A(i)j. The whole i-th row (as a row vector) is denoted A(i), and the j-th column (as a column vector) is denoted Aj. Thus

A = (A1 A2 ⋯ Ad) = (A(1); A(2); …; A(m)).
  • A(i, j)the entry of A in row i, column j (first index = row).
  • A(i)jalternative notation for the same entry; superscript in parentheses = row.
  • A(i)the i-th row of A (a row vector).
  • Ajthe j-th column of A (a column vector).
A1.17

Operator Norm of a Matrix

For a matrix A ∈ Mm,d, its operator norm (induced by the Euclidean vector norm) is defined by

|A|op = max{ |Ax| : x ∈ ℝd, |x| = 1 }.

It measures the maximum factor by which A can stretch a unit vector. Equivalently, |A|op is the largest singular value of A.

  • |A|opthe operator norm of A (induced by the Euclidean norm).
A1.18

Symmetric, PSD, and PD Matrices

The set of d × d real symmetric matrices (satisfying AT = A) is denoted Sd. Two important subsets are defined by the sign of the quadratic form xT A x:

  • Positive semi‑definite (PSD) matrices: S+d = { A ∈ Sd ∣ xT A x ≥ 0 for all x ∈ ℝd }.
  • Positive definite (PD) matrices: S++d = { A ∈ Sd ∣ xT A x > 0 for all non‑zero x ∈ ℝd }.

Every A ∈ S++d is invertible, but the converse is false.

  • Sdthe set of d × d real symmetric matrices.
  • S+dthe set of positive semi‑definite d × d matrices.
  • S++dthe set of positive definite d × d matrices.
A1.19

Orthogonal and Rotation Matrices

For m ≤ d, the set Om,d consists of all m × d real matrices A whose rows are orthonormal, i.e.,

A AT = Im.

When m = d, we write Od for the set of orthogonal matrices (square matrices with orthonormal rows and columns). A matrix A ∈ Od satisfies AT = A−1 and has determinant ±1.

The special orthogonal group SOd is the subset of Od consisting of orthogonal matrices with determinant exactly 1; these are precisely the rotation matrices in ℝd.

  • Om,d (m ≤ d)set of m × d matrices with orthonormal rows (A AT = Im).
  • Odthe orthogonal group: d × d matrices with AT A = Id.
  • SOdthe special orthogonal group: { A ∈ Od ∣ det(A) = 1 }, i.e., rotation matrices.
A1.20

Multilinear Map

A k‑linear map is a function a : (ℝd)k → ℝq that is linear in each of its k arguments. That is, for every index i = 1,…,k and for all vectors x1, …, xi−1, xi+1, …, xk held fixed, the map

xi ↦ a(x1, …, xi−1, xi, xi+1, …, xk)

is linear from ℝd to ℝq. Thus for scalars α, β and vectors u, v ∈ ℝd,

a(…, α u + β v, …) = α a(…, u, …) + β a(…, v, …).
  • aa k‑linear map from (ℝd)k to ℝq.
  • kthe number of arguments (arity) of the multilinear map.
  • qdimension of the target space.
A1.21

Symmetric Multilinear Map

A k‑linear map a is called symmetric if its value remains unchanged after any permutation of its arguments. Formally, for any permutation σ of {1,…,k},

a(xσ(1), …, xσ(k)) = a(x1, …, xk).
  • σa permutation of the indices {1,…,k}.
A1.22

Bilinear Form

When k = 2 (two arguments) and q = 1 (scalar‑valued), a 2‑linear map is called a bilinear form. It is a function a : ℝd × ℝd → ℝ that is linear in each argument separately.

  • bilinear forma 2‑linear map with target space ℝ (q=1).
A1.23

Norm of a k‑Linear Map

The norm of a k‑linear map a : (ℝd)k → ℝq is defined by

|a| = max{ |a(x1, …, xk)| : |xj| ≤ 1, j = 1,…,k }.

With this definition, the following inequality holds for all x1, …, xk ∈ ℝd:

|a(x1, …, xk)| ≤ |a| ∏j=1k |xj|.
  • |a|the norm of the k‑linear map a.
  • ∏j=1k |xj|product of the Euclidean norms of the k arguments.
A1.24

Positive Semidefinite / Definite Bilinear Form

A symmetric bilinear form a : ℝd × ℝd → ℝ is called

  • positive semidefinite if a(x, x) ≥ 0 for all x ∈ ℝd;
  • positive definite if it is positive semidefinite and, additionally, a(x, x) = 0 if and only if x = 0.

Analogous definitions hold for negative semidefinite and negative definite forms (by reversing the inequalities).

  • a(x, x)the quadratic form induced by a.
A1.25

Matrix Representation of Symmetric Bilinear Forms

Every symmetric bilinear form a : ℝd × ℝd → ℝ can be expressed in terms of a symmetric matrix A ∈ Sd as

a(x, y) = xT A y.

Moreover, a is positive semidefinite (resp. positive definite) if and only if the representing matrix A is positive semidefinite (resp. positive definite).

  • A ∈ Sdthe symmetric matrix representing the bilinear form a.
A1.26

Matrix Ordering Notation

For symmetric matrices A, B ∈ Sd, we write

  • A ≻ 0 (or A ≻ 0d×d) to indicate that A is positive definite;
  • A ⪰ 0 to indicate that A is positive semidefinite.

Similarly, A ≻ B means A − B ≻ 0, and A ⪰ B means A − B ⪰ 0.

  • A ≻ 0A is positive definite.
  • A ⪰ 0A is positive semidefinite.
  • A ≻ Bequivalently, A − B ≻ 0.
  • A ⪰ Bequivalently, A − B ⪰ 0.
A1.27

Norm of a Bilinear Form vs Operator Norm of its Matrix

If a symmetric bilinear form is given by a(x, y) = xT A y with A ∈ Sd, then the norm of the bilinear form (as defined for k‑linear maps with k = 2, q = 1) coincides with the operator norm of the matrix A:

|a| = |A|op.

Thus the bilinear form norm inherits all properties of the operator norm on matrices.

  • |a|norm of the bilinear form a.
  • |A|opoperator norm of the matrix A.
A1.28

Open Ball

For a centre x ∈ ℝd and a radius r > 0, the open ball of radius r around x is the set

B(x, r) = { y ∈ ℝd : |y − x| < r }.

It consists of all points whose Euclidean distance from x is strictly less than r.

  • B(x, r)the open ball of centre x and radius r.
  • |y − x|Euclidean distance between y and x.
A1.29

Closed Ball

The closed ball with centre x ∈ ℝd and radius r > 0 is denoted B̄(x, r) (or sometimes B̅(x, r)) and defined by

B̄(x, r) = { y ∈ ℝd : |y − x| ≤ r }.

It contains all points whose distance from x is at most r.

  • B̄(x, r)the closed ball of centre x and radius r.
A1.30

Open Set

A subset U ⊆ ℝd is called open if for every x ∈ U there exists a radius r > 0 such that the whole open ball B(x, r) is contained in U:

∀ x ∈ U, ∃ r > 0, B(x, r) ⊆ U.

In other words, around every point of an open set you can fit a small ball that stays entirely inside the set.

A1.31

Closed Set and Complement

A subset Γ ⊆ ℝd is closed if its complement

Γc = { x ∈ ℝd : x ∉ Γ }

is open. Equivalently, Γ is closed if it contains all its limit points (see closure).

  • Γcthe complement of set Γ in ℝd.
A1.32

Interior of a Set

The interior of a set A ⊆ ℝd, denoted Å or int(A), is the largest open set contained in A. Equivalently, a point x belongs to Å if and only if there exists r > 0 such that the open ball B(x, r) is entirely inside A:

x ∈ Å ⇔ ∃ r > 0, B(x, r) ⊆ A.
  • Å, int(A)the topological interior of A.
A1.33

Closure of a Set

The closure of a set A ⊆ ℝd, denoted Ā or cl(A), is the smallest closed set that contains A. A point x belongs to the closure if and only if every open ball around x intersects A:

x ∈ Ā ⇔ ∀ r > 0, B(x, r) ∩ A ≠ ∅.

Equivalently, x ∈ Ā if and only if there exists a sequence (xk) of points in A that converges to x (i.e., |xk − x| → 0 as k → ∞).

  • Ā, cl(A)the topological closure of A.
  • B(x, r)open ball of radius r around x (A1.28).
A1.34

Compact Set in ℝd

A set Γ ⊆ ℝd is called compact if it satisfies either of the following equivalent properties:

  • Sequential compactness: every sequence (xk) of points in Γ contains a subsequence that converges to some point in Γ.
  • Open‑cover compactness: whenever Γ is covered by a collection of open sets, there exists a finite subcollection that still covers Γ.

For subsets of ℝd (with the usual Euclidean distance), a set is compact if and only if it is both closed and bounded. This fact is known as the Heine–Borel theorem.

  • Γa subset of ℝd.
  • (xk)a sequence of points in Γ.
  • subsequencea sequence obtained by dropping some terms of the original sequence.
  • convergesapproaches a limit in ℝd (distance tends to 0).
A1.35

Metric Space

A metric space is a set B together with a distance function (or metric) ρ : B × B → [0, +∞) that satisfies the following three axioms for all x, y, z ∈ B:

ρ(x, y) = 0 ⇔ x = y,                (1.2a)
ρ(x, y) = ρ(y, x),                   (1.2b)
ρ(x, z) ≤ ρ(x, y) + ρ(y, z).              (1.2c)

Axiom (1.2c) is called the triangle inequality. The standard distance on ℝd is given by the Euclidean norm of the difference: ρ(x, y) = |x − y|.

  • Bthe underlying set of the metric space.
  • ρthe distance function (metric) on B.
  • ρ(x, y)distance between x and y (non‑negative real number).
  • triangle inequalityρ(x, z) ≤ ρ(x, y) + ρ(y, z).
A1.36

Open Sets, Closed Sets, and Convergence in Metric Spaces

In a metric space (B, ρ), the definitions of open and closed sets are the same as in ℝd, with the distance ρ replacing the Euclidean norm |x − y|.

A subset U ⊆ B is open if for every x ∈ U there exists r > 0 such that the open ball Bρ(x, r) = { y ∈ B : ρ(x, y) < r } is contained in U. A subset is closed if its complement in B is open.

A sequence (xn) in B converges to a point x ∈ B, written xn → x, if

ρ(xn, x) → 0 as n → ∞.

In other words, the distance between the sequence elements and the limit eventually becomes arbitrarily small.

  • Bρ(x, r)open ball in metric space (B, ρ) of radius r around x.
  • (xn)a sequence in B.
  • xn → xthe sequence converges to x (ρ(xn, x) → 0).
A1.37

Compact Sets in General Metric Spaces

In an arbitrary metric space, a subset K ⊆ B is called compact if it satisfies the same sequential or open‑cover definitions as in ℝd:

  • Every sequence in K has a subsequence converging to a point in K (sequentially compact).
  • Every open cover of K admits a finite subcover (compact by open covers).

In a general metric space these two notions are equivalent. However, unlike ℝd, a set that is closed and bounded is not necessarily compact; the Heine–Borel theorem holds only in certain spaces (e.g., ℝd with the usual metric).

  • Ka subset of a metric space B.
  • compact (in metric space)sequentially compact or open‑cover compact (equivalent in metric spaces).
